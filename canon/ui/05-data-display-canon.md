# CRM OS Data Display Canon

## Purpose

DataTable is the most critical CRM OS UI primitive.

## Required Capabilities

- Column definitions
- Sort
- Filter
- Search
- Saved views
- Bulk actions
- Pagination
- Export
- Empty/loading/error states
- Permission-aware actions
- AI-assisted table actions

## Table Rules

- Sticky header required.
- Row height standard: 48px.
- Compact and comfortable density modes.
- Financial values right-aligned.
- Status must use labels, not color alone.
- Export must respect permissions.
- Search must not leak restricted records.

## Domain Tables

- Customers
- Leads
- Opportunities
- Quotes
- Orders
- Products
- Inventory
- Tickets
- Audit Logs
- Users
- Reports
