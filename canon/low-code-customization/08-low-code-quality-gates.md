# CRM OS Low-Code Quality Gates

## Required Gates

- [ ] Custom fields are tenant-scoped.
- [ ] Custom modules cannot override system modules.
- [ ] Metadata changes are audited.
- [ ] Field-level permissions are enforced.
- [ ] Validation rules run server-side.
- [ ] Custom views respect permissions.
- [ ] No arbitrary code execution exists.
- [ ] No raw SQL execution exists.
- [ ] AI-suggested customization requires review.
- [ ] Low-code metadata supports export/import safely.
