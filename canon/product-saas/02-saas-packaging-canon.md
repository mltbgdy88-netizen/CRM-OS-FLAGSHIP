# CRM OS SaaS Packaging Canon

## Package Model

CRM OS supports modular SaaS packaging.

Recommended package tiers:

1. Starter
2. Professional
3. Business
4. Enterprise
5. Enterprise Plus

## Module Packaging

Core modules:

- CRM Core
- Lead Management
- Pipeline Management
- Quote Management
- Task & Activity
- Dashboard

Business modules:

- Order Management
- Product Catalog
- Inventory
- Finance Lite
- Payments & Receivables
- Workflow
- Customer Portal

Enterprise modules:

- Omnichannel Inbox
- Support & SLA
- Dealer Portal
- Public API
- Webhooks
- SSO/SAML
- Audit & Compliance
- Advanced Security
- AI Gateway
- Ask CRM
- Prediction
- Advanced Analytics
- Report Builder
- Data Quality

## Quotas

Quota dimensions:

- Users
- Tenants/workspaces
- Branches
- Dealers
- Storage
- API calls
- Webhook deliveries
- AI credits
- Messages
- Integrations
- Reports
- Automations

## Feature Flags

Every paid module and risky feature must be feature-flagged.
