# CRM OS Mobile Strategy Canon

## Purpose

CRM OS mobile is a companion experience, not a full desktop replacement.

## Primary Mobile Use Cases

- Field sales customer lookup
- Lead capture
- Visit planning
- Task execution
- Quote/order quick review
- Stock availability check
- Approval decisions
- Notification triage
- Customer communication
- Offline data capture

## Product Rule

Desktop remains the primary enterprise control center. Mobile focuses on action, review, capture, and follow-up.
