# CRM OS Field Sales Canon

## Core Field Sales Flow

1. Open daily plan
2. View assigned customers and tasks
3. Navigate to customer
4. Check in
5. Capture visit note
6. Add photos/files if needed
7. Create lead/opportunity/task
8. Check stock or product catalog
9. Send follow-up
10. Check out
11. Sync updates

## Required Components

- MobileDailyPlan
- MobileCustomerCard
- MobileVisitCheckIn
- MobileVisitNoteComposer
- MobileTaskList
- MobileRouteMap
- MobileActivityCapture
- MobileSyncStatus
- MobileOfflineBanner

## Audit Requirements

- Check-in/check-out timestamp
- Optional geo coordinates based on policy
- Actor id
- Device id
- Tenant id
- Sync timestamp
