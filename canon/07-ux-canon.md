# CRM OS UX Canon v1

## UX Identity

CRM OS is:

```text
Dark First
Enterprise Grade
Information Dense
AI Native
Fast Navigation
Command Center Experience
```

## Design Inspirations

- Linear
- Stripe Dashboard
- Attio CRM
- Notion
- Vercel

## Design System

- Typography: Inter
- Grid: 8px system
- Radius: 8 / 12 / 16 / 24
- Primary accent: Blue
- AI accent: Purple
- Data density: high but readable

## App Shell

Authenticated CRM OS screens use:

```text
Top Command Bar
Sidebar
Main Content
Optional Right AI Panel
```

## Required UI Systems

- AppShell
- Sidebar
- Topbar
- Command Palette
- Global Search
- Notification Center
- DataTable
- FilterBar
- Saved Views
- Form System
- Drawer System
- Modal System
- Timeline
- Kanban Board
- AI Panel
- Dashboard Cards

## Permission-Aware UI

Unauthorized actions must be hidden or shown with safe explanation. UI must not leak restricted data.

## AI UX

AI must appear as an auditable assistant, not an autonomous hidden actor. AI actions require approve/edit/reject controls where they affect data.
