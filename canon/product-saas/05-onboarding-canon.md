# CRM OS Onboarding Canon

## Onboarding Goals

The onboarding flow must help a new tenant reach the first successful CRM workflow quickly.

Target first-value path:

Create Tenant → Invite User → Create Customer → Create Lead → Convert Lead → Create Opportunity → Create Quote → View Dashboard

## Required Onboarding Steps

1. Tenant profile setup
2. Workspace settings
3. Currency and timezone
4. User invitation
5. Role selection
6. Sample data option
7. Customer import option
8. Product catalog setup option
9. Pipeline setup
10. First dashboard view

## Demo Data

Demo data must be tenant-scoped and removable.

## Import

Import wizard must support:

- CSV
- Excel
- Field mapping
- Duplicate detection
- Validation preview
- Error report
