# CRM OS Low-Code Security Canon

## Mandatory Controls

- Tenant isolation
- RBAC enforcement
- Field-level permissions
- Audit logging
- Input validation
- Rate limits for metadata changes
- Export restrictions
- Safe rendering for custom labels/help text

## AI and Low-Code

AI may suggest custom fields, forms, workflows, or dashboards, but must not apply them silently.

All AI-generated customization changes require approve/edit/reject review.
