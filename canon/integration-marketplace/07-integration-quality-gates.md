# CRM OS Integration Quality Gates

## Gates
- API contract exists before implementation.
- Webhook payload schema exists.
- Signature verification test exists.
- Rate limit test exists.
- Tenant isolation test exists.
- Permission scope test exists.
- Retry and failure tests exist.
- Integration health state exists.
- Audit log coverage exists.

## Definition of Done
An integration is not done until it can be installed, tested, monitored, disabled and audited.
