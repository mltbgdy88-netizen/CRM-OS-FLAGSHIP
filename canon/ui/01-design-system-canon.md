# CRM OS Design System Canon

## Purpose

CRM OS uses a dark-first, enterprise-grade, information-dense design language.

## Visual Identity

- Dark Graphite base
- Primary Blue: `#3B82F6`
- AI Purple: `#8B5CF6`
- Typography: Inter / system-ui
- Grid: 8px
- Radius: 16px standard
- Density: enterprise high-density without clutter

## Principles

1. Information first.
2. AI-native but auditable.
3. Permission-aware UI.
4. Reusable components before screens.
5. Dark-first consistency.
6. No generic admin template look.
7. No decorative dashboard noise.

## Token Categories

- Color
- Typography
- Spacing
- Radius
- Elevation
- Border
- Opacity
- Z-index
- Breakpoint
- Motion

## Hard Rules

- No raw colors in components.
- No arbitrary spacing unless approved.
- No screen-specific design systems.
- No duplicate component variants.
- All states must be represented: default, loading, empty, error, permission-restricted, read-only.
