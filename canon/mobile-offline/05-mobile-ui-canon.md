# CRM OS Mobile UI Canon

## UI Principle

Mobile is action-first, dense enough for work, but simplified compared to desktop.

## Navigation

- Bottom navigation for primary mobile modules
- Drawer for secondary settings
- Search always accessible
- Notifications accessible from top bar
- AI assistant accessible but controlled

## Mobile Priority Screens

- Today
- Tasks
- Customers
- Leads
- Approvals
- Notifications
- Search
- Product Catalog
- Offline Sync Center

## Responsive Rule

Do not force desktop data tables into mobile. Use list/card views with drilldown.
