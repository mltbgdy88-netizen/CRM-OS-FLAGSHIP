# CRM OS Business Rules & Validation Canon

## Business Rule Types

- required field rule
- conditional visibility rule
- numeric threshold rule
- date window rule
- duplicate detection rule
- approval required rule
- notification trigger rule
- workflow trigger rule

## Execution Rules

- Rules must run server-side for enforcement.
- Client-side rules are usability helpers only.
- Rules must be versioned.
- Rule execution must be auditable when it affects data changes.
