# CRM OS Customer Segment Canon

## Primary Customer Segments

CRM OS is designed for:

- B2B sales organizations
- Distributor and dealer networks
- Retail + wholesale hybrid companies
- Decoration, wallpaper, textile, furniture, and product catalog businesses
- Companies needing quote/order/inventory linkage
- Businesses needing omnichannel customer operations
- SaaS-ready multi-branch organizations

## User Personas

Core personas:

- Tenant Admin
- Sales Manager
- Sales Representative
- Finance User
- Warehouse User
- Support Agent
- Marketing User
- Dealer User
- Customer Portal User
- Executive User
- System Admin

## Persona Priority

Initial product priority:

1. Tenant Admin
2. Sales Manager
3. Sales Representative
4. Warehouse User
5. Finance User
6. Support Agent

8. Executive User
