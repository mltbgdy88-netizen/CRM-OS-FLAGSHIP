# CRM OS Custom Forms & Views Canon

## Form Customization

Allowed:
- field ordering
- section grouping
- required field rules
- visibility rules
- read-only rules
- conditional display
- helper text
- default values

Forbidden:
- removing required system fields
- hiding audit/security warnings
- bypassing approval logic
- editing tenant_id or system ownership fields

## View Customization

Allowed:
- saved views
- table columns
- filters
- sort order
- density
- grouped views
- Kanban views where supported

All custom views must remain permission-aware.
